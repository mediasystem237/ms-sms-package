# ms-sms-package

`ms-sms-package` est un package PHP simple et efficace pour envoyer des SMS en utilisant l'API de NEXAH. Ce package permet d'intégrer facilement la fonctionnalité d'envoi de SMS dans vos applications PHP.

## Installation

Pour installer le package, utilisez Composer. Si vous n'avez pas encore installé Composer, vous pouvez le télécharger et l'installer depuis [getcomposer.org](https://getcomposer.org/).

Ensuite, exécutez la commande suivante dans le répertoire de votre projet :

```bash
composer require mediasystem237/ms-sms-package
